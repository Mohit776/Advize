# **App Name**: Advizor

## Core Features:

- Creator Dashboard: Personalized dashboard for creators to track campaigns, earnings, and analytics.
- Business Dashboard: Analytics dashboard for businesses to monitor campaign performance, budget, and ROI.
- Campaign Marketplace: Centralized marketplace listing available campaigns with filtering and search options.
- User Authentication: Secure authentication system for creators and businesses with role-based access control.
- Profile Management: Profile pages for creators and businesses to manage personal information and campaign history.
- Campaign Creation: Form for businesses to launch new campaigns, set budget, requirements, and platform.
- SmartMatch: AI tool that recommends the best creators for a particular campaign, increasing the effectiveness of the ads served.

## Style Guidelines:

- Primary color: Subtle blue (#72BCD4) to represent trust and professionalism.
- Background color: Light gray (#F0F4F7) for a clean, modern look.
- Accent color: Deep blue (#3F51B5) for interactive elements and calls to action.
- Body font: 'PT Sans' for readability and a modern feel.
- Headline font: 'Space Grotesk' for a clear, professional style.
- Use line icons to complement the clean design.
- Responsive grid layout for consistent design across devices.
- Use smooth transitions for page navigations.
